# Proof Assets — `[brand-slug]`

> The receipts. Numbers, customers, quotes, awards, press, and screenshots the brand can cite publicly. The system relies on this file to avoid making up evidence.

---

## How this file is used

- The system **only** uses claims listed here.
- Each item is tagged `permission_to_cite_publicly: true | false`.
- If a slide needs evidence not listed here, the system writes `[PROOF_NEEDED: short description]` and surfaces it in Missing Inputs.
- Update this file every time you ship a post that earned a useful piece of feedback or whenever a customer says something quotable.

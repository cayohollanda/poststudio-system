## Voice review checklist

Before shipping any caption, run this 4-step check:

1. **Read it out loud.** If you stumble, rewrite.
2. **Search for words from the banlist.** If you find any, rewrite.
3. **Check sentence rhythm.** If 4+ consecutive sentences are the same length, rewrite.
4. **Compare to voice examples above.** If the new line feels like a different brand than the examples, rewrite.

# WeAura AI — Logo Pack v2.0

71 arquivos · PNG alta resolução + SVG source

## Estrutura

```
mark/      Aperture detalhada (com variação de opacidade) · para ≥64px
favicon/   Aperture flat (opacidade cheia, ponto maior) · para ≤256px
lockup/    Mark + wordmark "WeAura AI" em Manrope
svg/       SVG source de cada variante · use para qualquer tamanho arbitrário
```

## Guia de uso

### mark/
- `mark-primary-*.png` — Aperture creme + ponto volt · **padrão para backgrounds escuros** · fundo transparente
- `mark-dark-*.png` — Aperture escura + ponto volt · para backgrounds claros · fundo transparente
- `mark-primary-on-dark-*.png` — com fundo warm dark incluído
- `mark-dark-on-light-*.png` — com fundo warm light incluído
- `mark-dark-on-volt-*.png` — variante inversa sobre volt
- Tamanhos: 512, 1024, 2048, 4096 px

### favicon/
- `favicon-*.png` — versão creme sobre fundo dark (padrão)
- `favicon-light-*.png` — versão dark sobre fundo light
- `favicon-volt-*.png` — versão dark sobre fundo volt
- `favicon-transparent-*.png` — transparente
- Tamanhos: 16, 32, 48, 64, 128, 180 (Apple touch), 256, 512 px

### lockup/
- `lockup-on-dark-*` — lockup completo com fundo dark
- `lockup-on-light-*` — lockup com fundo light
- `lockup-on-volt-*` — lockup com fundo volt (dot preto, sem volt no mark)
- `lockup-transparent-*` — sem fundo, creme ou dark
- Alturas: 600 e 1200 px (proporção ~2.64:1)

# Arguments — Pattern Library

> Patterns for showing the mechanism. The "why-this-works" slide is what earns the rest of the carousel.

---

## Mechanism patterns

### Structural-not-behavioral
```
[GROUP_THAT_SUCCEEDS] don't have more [TRAIT]. They have fewer [SURFACE/OBSTACLE]. The advantage is structural, not behavioral.
```

### First-principles
```
If [PROCESS] is [METAPHOR], then every [STEP] is a [SUB-METAPHOR]. The minimum-viable version is [MIN_FORM]. Anything else is overhead.
```

### Cost-revealed
```
The default looks free. It's not. Every [DEFAULT_BEHAVIOR] costs [METRIC] per [TIMEFRAME]. Multiply by [SCALE].
```

### Threshold-crossed
```
Below [THRESHOLD], [APPROACH_A] works. Above [THRESHOLD], it doesn't. [AUDIENCE] crossed [THRESHOLD] when [TRIGGER]. They need [APPROACH_B].
```

### Asymmetry
```
[PROBLEM] takes [TIMEFRAME] to create. The fix takes [SHORT_TIMEFRAME]. The asymmetry is the lever.
```

### Compound effect
```
Each [SMALL_DECISION] feels free. The cost compounds. After [TIMEFRAME] of small decisions, [BIG_OUTCOME] is inevitable.
```

### The unspoken default
```
Nobody schedules [DEFAULT_PROBLEM]. It happens because [SYSTEM] doesn't include [STRUCTURE]. Add the structure; the default disappears.
```

---

## How to write a strong argument slide

1. **Name the assumed cause.** ("Most people think X causes Y.")
2. **Reveal the actual cause.** ("Actually, Z causes Y.")
3. **Show why.** ("Because [structural reason].")
4. **Implication.** ("Which means [actionable shift].")

Compress to one slide if the argument fits. Spread across two slides if it needs more room.

---

## Anti-patterns

- **Argument by assertion.** "X works because it just works." → fail.
- **Argument by authority.** "[Expert] says X." → only if you can quote them verbatim with permission.
- **Argument by inevitability.** "Everyone knows X." → vague, not defensible.
- **Argument by analogy alone.** Analogies are illustrations, not proofs. Pair with mechanism.

---

## Mechanism length

- One slide is ideal.
- Two slides if the mechanism has two sub-points.
- Never spread across three slides — the audience disengages.

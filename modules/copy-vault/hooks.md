# Hooks — Pattern Library

> First-slide patterns. Pick by angle. Substitute placeholders. Filter through Brand Pack voice.

---

## Status-quo callout

```
Most [AUDIENCE] still believe [COMMON_BELIEF]. That's why they're stuck at [PROBLEM].
```

```
[N]% of [AUDIENCE] still default to [BEHAVIOR]. The fix takes [TIMEFRAME].
```

## Banned hook patterns (do not generate)

- "Are you tired of...?"
- "Did you know that...?"
- "Hot take:"
- "Unpopular opinion:"
- "Buckle up." / "Strap in."
- "Hey [AUDIENCE]!"
- All-caps hooks ("THE ONE THING...")
- Emoji-only hooks (🔥🚀💯)
- Listicle ("5 ways to...") unless the Brand Pack examples confirm they work for this audience

---

## Length rule

- Target: ≤ 14 words.
- Maximum: 18 words for a longer-form audience (LinkedIn essay-style).
- Below 7 words risks ambiguity; below 5 is rare.

---

## Specificity test

Run every hook through:

| Specificity level | Example |
|---|---|
| 1 (vague) | "Get more done in less time." |
| 2 (slightly specific) | "Save time on meetings." |
| 3 (audience-specific) | "Most engineers waste 3 hours/week on standup." |
| 4 (audience + mechanism) | "Most engineers waste 3 hours/week on standup. The fix takes 5 minutes." |
| 5 (audience + mechanism + tension) | "Most engineers waste 3 hours/week on standup. The fix takes 5 minutes — and your manager will hate it." |

Aim for level 4 or 5. Level 1-2 = rewrite.

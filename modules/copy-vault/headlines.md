# Headlines — Pattern Library

> Slide-level headline patterns. Different from hooks (hooks are slide 1 only). Headlines run across slides 2-N and the CTA slide.

---

## By role

### Problem slide

```
At [TIMEFRAME], [SYMPTOM] became unavoidable.
```

```
[AUDIENCE]'s week stops working when [TRIGGER].
```

```
The [PROCESS] you used to ship in [SHORT_TIME] now takes [LONG_TIME].
```

### Mistake slide

```
I kept [WRONG_BEHAVIOR] to fix the symptoms.
```

```
The default move: [DEFAULT_ACTION]. The cost: [METRIC].
```

```
Most [AUDIENCE] respond by [WRONG_RESPONSE]. It compounds the problem.
```

### Reframe slide

```
The problem wasn't [SUSPECTED_CAUSE]. It was [REAL_CAUSE].
```

```
[CATEGORY] isn't a [OLD_FRAME] problem. It's a [NEW_FRAME] problem.
```

```
Stop fighting [SYMPTOM]. Replace [STRUCTURE].
```

### Mechanism slide

```
[GROUP_THAT_WINS] don't have more [TRAIT]. They have fewer [SURFACE_OR_OBSTACLE].
```

```
The senior-operator advantage is structural, not behavioral.
```

```
[OUTCOME] is downstream of [LEVER]. Pull that.
```

### Example / Proof slide

```
Here's what changed when [PERSON/TEAM] applied [APPROACH].
```

```
Before: [BEFORE]. After: [AFTER]. Difference: [INTERVENTION].
```

### Application slide

```
What I removed first.
```

```
The 3 surfaces to kill this week.
```

```
Start with the smallest move that compounds.
```

### CTA slide

```
Save this for the next time you're about to [DEFAULT_BEHAVIOR].
```

```
Drop your [THING] in the comments. I'll send the [BONUS].
```

```
DM me '[KEYWORD]' for the [RESOURCE].
```

```
Try [PRODUCT] free at [URL]. Link in bio.
```

```
Follow for more [TOPIC] without [PAIN].
```

---

## Length

- Target: 6-12 words.
- Maximum: 14 words.
- Anything longer = blog copy.

---

## Active voice rule

Every headline in active voice. "We rebuilt the onboarding" beats "The onboarding was rebuilt."

---

## Banned

- "X tips that will change your life" / similar.
- "Here's the thing about..."
- "Imagine if..."
- "Picture this:"
- "Let's dive in."
- "You won't believe..."
- All-caps emphasis on every other word.

---

## Multi-line headlines

If a headline truly needs two lines (rare), break on a natural phrase boundary, never mid-clause.

```
Most AI tools quietly destroy your deep work.
And the chat is why.
```

The second line goes in `support_text` if it's < 8 words; otherwise it's its own slide.

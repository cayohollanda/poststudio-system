# CTAs — Pattern Library

> One CTA per post. Match energy to angle.

---

## Save

```
Save this for the next time you're about to [DEFAULT_BEHAVIOR].
```
```
Save this. Send it to [AUDIENCE] making this call.
```
```
Save this for [DECISION_MOMENT].
```

## Share

```
Send this to [AUDIENCE_DESCRIPTION] making this mistake.
```
```
Share with the [ROLE] on your team who needs to read it.
```

## Comment

```
Drop your [THING] in the comments — I want to compare notes.
```
```
Comment '[KEYWORD]' if you've shipped [SCENARIO].
```
```
What's the one [TOOL/PROCESS/HABIT] you'd kill in your stack tomorrow?
```
```
Tell me a [THING] I missed in the comments.
```

## Click

```
Full [GUIDE/DATA/REPORT] at the link in bio.
```
```
The complete [TEMPLATE/CHECKLIST] is at [URL].
```

## DM

```
DM me '[KEYWORD]' and I'll send you the [RESOURCE].
```
```
DM me your [SITUATION] — I'll send the relevant [TEMPLATE].
```

## Follow

```
Follow for more [TOPIC] without the [PAIN_POINT].
```
```
Follow if [REASON_THAT_FITS_BRAND].
```

## Try

```
Try [PRODUCT] free at [URL].
```
```
[PRODUCT] is free for [USE_CASE]. Try it at [URL].
```

---

## CTA matching by angle

| Angle | Best CTAs |
|---|---|
| Contrarian | Comment, Share |
| Mistake | Comment, Save |
| Hidden Feature | Save, Try |
| Before/After | Save, DM |
| Comparison | Save, Comment |
| Myth vs Reality | Share, Save |
| Benchmark | Click, DM |
| Build in Public | Comment, Follow |
| Founder Perspective | Comment, Follow |
| Technical Breakdown | Save, Click |
| Workflow Reveal | Save, DM |
| Checklist | Save, Share |
| Opinionated Ranking | Comment, Share |
| You're Using X Wrong | Save, Comment |
| Stop Doing X | Comment, Save |
| Nobody Tells You | Save, Share |
| I Tested X | Click, DM |

---

## Banned CTAs

- "Hope this helps!"
- "Let me know your thoughts."
- "Like and follow for more!"
- "What did I miss?"
- "Drop a 🔥 if you agree"
- "Comment with your favorite emoji"
- "Tag a friend who needs this" (overused; reads as engagement bait)

---

## Rule

One CTA per post. Never two. Never three. The audience tunes out multi-CTA stacks.

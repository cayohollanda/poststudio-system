# Captions — Pattern Library

> Caption opener and closing patterns. Captions hold the conversation; the slides held the argument.

---

## Opener patterns

### Restate the hook differently
```
[Restated hook in different words.]
[Strongest single sentence from the carousel.]
```

### Confession opener
```
I [DID_THING] for [TIMEFRAME] before I figured out [INSIGHT].
[Then the line that earned the post.]
```

### Number-led opener
```
[N] [THINGS] tested. [M] [SURVIVORS]. Here's what shipped.
```

### Question opener (rare; only when audience-specific)
```
[Question naming the exact decision the audience is making.]
```

---

## Body patterns

### Argument-with-receipt
```
[Setup — the situation everyone recognizes.]
[Opinion — the brand's stance.]
[Mechanism — why the stance is true.]
[Receipt — the example or proof.]
[Reframe — the principle, generalized.]
```

### Story-with-takeaway
```
[Specific anecdote in 2-3 sentences.]
[The lesson.]
[The principle, generalized.]
```

---

## Closing patterns

### Conversation prompt
```
Tell me — [specific question targeting a decision].
```
```
What's the one [THING] you'd [VERB] tomorrow if you could?
```
```
If you've shipped [THING], what would you tell someone starting?
```

### Save / Share trigger
```
Save this for the next time you're about to [BEHAVIOR].
```

### CTA finish
```
[Single CTA on its own line.]

— [Founder name / signature]
```

## Length per platform

| Platform | Word count |
|---|---|
| LinkedIn | 150-250 |
| Instagram | 100-180 |
| TikTok | 80-140 |
| X (post) | 100-140 (or 5-7 tweet thread for longer) |
| Threads | 100-200 |

---

## Banned

- "Hope this helps!"
- "Let me know your thoughts."
- "Drop a 🔥 if you agree"
- "Tag a friend who needs this" (engagement bait)
- Hashtag soup at the end (3-5 brand-pack-approved hashtags max)
- Emoji in every line
- "Hey [audience]!" openers
- "Buckle up" / "Strap in"

---

## Comment-prompt rules

A good comment prompt:

- Names a specific decision the audience is making.
- Asks for a reaction, not approval.
- Implies the answer is short (one line) so the bar to reply is low.
- Doesn't ask for praise.

A bad comment prompt:

- "Thoughts?" → too low-effort.
- "Agree or disagree?" → false binary.
- "Did this help?" → asks for praise.
- "Drop a 🔥" → engagement bait.

# Script Structures

> Beat structures by duration. Pick the one that fits the duration and topic.

---

## 15-second structure (30-40 words)

```
0:00-0:02  Hook — visual + on-screen text + voice
0:02-0:12  One beat (problem + solution compressed)
0:12-0:15  CTA
```

Use when: high-frequency content, simple takeaway, single-mechanism reveal.

---

## 30-second structure (65-80 words) — workhorse

```
0:00-0:02  Hook
0:02-0:05  Set-up (tension)
0:05-0:18  Body (2 beats: problem + solution, OR 2-3 steps)
0:18-0:24  Payoff
0:24-0:30  CTA
```

Use when: standard short-form post.

---

## 45-second structure (95-115 words)

```
0:00-0:02  Hook
0:02-0:05  Set-up
0:05-0:12  Beat 1 (problem context)
0:12-0:22  Beat 2 (the move)
0:22-0:32  Beat 3 (proof or example)
0:32-0:40  Payoff (the principle)
0:40-0:45  CTA
```

Use when: tutorial, mini-essay, founder reflection.

---

## 60-second structure (130-160 words)

```
0:00-0:02  Hook
0:02-0:08  Set-up
0:08-0:25  Beat 1 (the situation in detail)
0:25-0:42  Beat 2 (the mechanism)
0:42-0:54  Beat 3 (proof or application)
0:54-1:00  CTA
```

Use when: founder essay, deep tutorial.

---

## 90-second structure (200-240 words)

```
0:00-0:02  Hook
0:02-0:10  Set-up + premise
0:10-0:30  Beat 1
0:30-0:55  Beat 2
0:55-1:20  Beat 3
1:20-1:30  CTA
```

Use rarely. Only when the audience is patient and the topic deserves it.

---

## Per-type beat priorities

### Talking-head
Voice carries the script. On-screen text reinforces.

### Faceless
On-screen text carries the script. Voice (if any) is supportive.

### B-roll-led
Visual sequence carries the script. On-screen text + voice over each cut.

### Hybrid
Founder talks for ~50%; B-roll cuts cover the rest.

---

## Cross-duration rule

Whatever duration you pick, the **first 2 seconds are non-negotiable**: pattern interrupt + promise visible (sound-off).

If the audience scrolls in the first 2 seconds, the rest doesn't matter.

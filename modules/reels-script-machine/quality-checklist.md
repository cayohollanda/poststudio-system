# Quality Checklist — Reels Script Machine

> Short-video-specific checks. Run before returning.

---

## Critical (any fail → rewrite)

- [ ] Hook visible in first 2 seconds.
- [ ] Sound-off legible (on-screen text covers the message even if audio is muted).
- [ ] One idea per beat.
- [ ] Word count fits duration (per `system/reels-framework.md`).
- [ ] One CTA — not multi-action stacks.
- [ ] No invented claims.
- [ ] Voice matches Brand Pack.
- [ ] B-roll directions are concrete (a director could shoot them).
- [ ] On-screen text ≤6 words per block.
- [ ] Caption hooks line 1 (not "today I want to talk about").

## Quality (each fail → flag)

- [ ] No "Hey guys" / "What's up everyone" openers.
- [ ] No "today I want to talk about..." burning the first 3 seconds.
- [ ] No logo intro card eating the first second.
- [ ] B-roll cuts vary across the script (no 4 cuts of "hands working at laptop").
- [ ] On-screen text 1+ second per word minimum legibility.
- [ ] Editing notes capture non-obvious post-production decisions.
- [ ] Caption length fits platform.
- [ ] Hashtag count: 3-5 max, brand-pack-approved only.
- [ ] CTA matches platform (TikTok rewards comment / save; Reels rewards save / DM; YouTube Shorts rewards click).
- [ ] No banned vocabulary from Brand Pack `voice.md`.

---

## Anti-patterns

- **Hook lives in beat 2.** "The real opening" comes after a "today we're going to talk about" intro → kill the intro.
- **Walls of on-screen text.** ≥10 words per block → split.
- **B-roll mismatch.** Same cut idea appears 4 times → vary.
- **Multi-CTA confusion.** "Save and follow and DM and comment" → pick one.
- **Caption-as-script.** Caption transcribes the audio → caption is a separate beat with new value.
- **Music-only intro.** Slow zoom into a face with no on-screen text → audience scrolls.

---

## Approval

A reels script passes when:
- All critical pass.
- ≥ 7 of 10 quality items pass.
- The first 2 seconds are unmistakably a scroll-stop.

If it fails the first-2-seconds test, rewrite the hook before anything else.
